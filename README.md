# Yossarian

This app's purpose is to improve intercompany communication and documentation. In fact it aims to become an easy-to-use intranet solution. It should cover the following:

- News
- Knowledge base
- Calendar 
- Project management
- Customer relations

Quite ambitious. Yossarian was quite far along, until I decided to completely start over.
